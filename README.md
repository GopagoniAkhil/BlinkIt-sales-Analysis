## 📌 Project Overview

This project focuses on analyzing Blinkit's retail sales dataset (8,523 records) to uncover revenue patterns, outlet performance insights, and product contribution trends using Power BI.

The objective was to transform raw transactional data into an interactive and business-focused dashboard that supports data-driven decision-making.

---

## 🎯 Business Objective

Retail businesses generate large volumes of data daily. However, without proper analysis and visualization, identifying meaningful insights becomes challenging.

This project aims to:

- Analyze overall sales performance
- Identify top-performing outlet types
- Compare Tier 1, Tier 2, and Tier 3 sales distribution
- Evaluate product category contribution
- Study the impact of outlet size and visibility on sales
- Examine historical sales trends by establishment year

## 💡 Key Business Insights

- Tier 3 outlets generated the highest total revenue.
- Supermarket Type 1 contributed ₹787.5K in sales.
- Regular fat products accounted for 64.6% of overall sales.
- Medium-sized outlets demonstrated consistent performance.
- Sales peaked around 2018 based on establishment trends.
